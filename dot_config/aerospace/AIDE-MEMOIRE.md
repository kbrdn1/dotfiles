# 🎯 AIDE-MÉMOIRE AEROSPACE - ALT DROITE UNIQUEMENT

## 🔑 TOUCHE IMPORTANTE
```
┌─────────────────────────────────────────┐
│  [fn] [ctrl] [⌥] [⌘]   [⌘] [⌥→] [ctrl]  │
│                            ↑            │
│                      CETTE TOUCHE       │
│                      UNIQUEMENT         │
└─────────────────────────────────────────┘
```

## 🚀 RACCOURCIS ESSENTIELS

### 🏠 Navigation Workspaces
```
⌥→ + 1    💻 Développement
⌥→ + 2    🖥  Terminal  
⌥→ + 3    🌐 Web
⌥→ + 4    📧 Communication
⌥→ + 5    🤝 Réunions

⌥→ + 6    🎨 Design

⌥→ + 8    🔧 Utilitaires  
⌥→ + 9    🎵 Média
⌥→ + 0    🎮 Gaming

⌥→ + Tab  ↩️  Workspace précédent
```

### 🎯 Navigation Fenêtres (Vim Style)
```
       ⌥→ + K
         ↑
⌥→ + H ← ● → ⌥→ + L
         ↓
       ⌥→ + J
```

### ⚡ Actions Rapides
```
⌥→ + F        🔲 Plein écran
⌥→ + Space    🔄 Toggle floating/tiling
⌥→ + ⌃ + E    ⚖️  Équilibrer fenêtres
```

### 📦 Déplacements
```
⌥→ + ⇧ + H/J/K/L    Déplacer fenêtre ↑↓←→
⌥→ + ⇧ + 1-0        Déplacer vers workspace
⌥→ + ⇧ + S          Joindre en bas
⌥→ + ⇧ + B          Joindre à droite
```

### 🔧 Redimensionnement
```
⌥→ + ⌃ + H/J/K/L         -/+ 50px
⌥→ + ⌃ + ⇧ + H/J/K/L     -/+ 100px
```

### 📺 Multi-Moniteurs
```
⌥→ + ←/→/↑/↓              Focus moniteur
⌥→ + ⇧ + ←/→/↑/↓          Déplacer fenêtre
⌥→ + ⌃ + ←/→              Déplacer workspace
```

## 🛠 Maintenance
```
⌥→ + ⌃ + R              Recharger AeroSpace
⌥→ + ⇧ + ⌃ + R          Recharger skhd
⌥→ + ⌃ + M              Ouvrir gestionnaire
```

## 💡 TIPS RAPIDES

### ✅ Combos Magiques
- `⌥→ + 1` puis `⌥→ + ⌃ + C` → Dev workspace + VS Code
- `⌥→ + Tab` → Basculement rapide
- `⌥→ + ⌃ + E` → Tout équilibrer

### ❌ À Éviter
- **JAMAIS Alt gauche** - conflicts macOS
- **JAMAIS sans ⌥→** - pas de raccourcis AeroSpace

### 🎯 Workflow Type
```
1. ⌥→ + 1    → Workspace dev
2. ⌥→ + 2    → Workspace terminal  
3. ⌥→ + 3    → Workspace web
4. ⌥→ + Tab  → Bascule rapide
```
